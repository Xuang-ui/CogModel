第一周的课程中，我们引入了经典的 Rescorla-Wagner模型（以下简称为RW模型），并通过手动模拟演示了生物体如何在逐个试次中更新条件刺激（CS，铃声与灯光）与无条件刺激（US，食物）之间的联结强度。

> 如需回顾，可参考第一周的课件，或查阅 Rescorla, R. A., & Wagner, A. R.（1972）发表于 Black, A. H., & Prokasy, W. F.（主编）*Classical Conditioning II: Current Research and Theory* 第三章的文章 "A theory of Pavlovian conditioning: Variations in the effectiveness of reinforcement and nonreinforcement"。

本次作业将探讨一个更复杂的实验情境。实验共设四种基本刺激：两种视觉刺激（记为A和B）与两种听觉刺激（记为X和Y）。每个试次中，一种视觉刺激与一种听觉刺激配对呈现，构成四种复合条件刺激（AX、AY、BX、BY）；同时，以一定概率给予动物食物奖赏；实验者记录动物单位时间内的啄食频次作为条件反应的行为指标。实验分为学习阶段（前360个试次）和测验阶段（后40个试次），共400个试次。

实验数据记录在`data.csv`中：

| 列名         | 说明                            |
| :--------- | :---------------------------- |
| `Phase`    | 实验阶段（Learning=学习阶段，Test=测验阶段） |
| `Stimulus` | 当前试次的复合刺激类型（AX、AY、BX 或 BY）    |
| `Light_A`  | 视觉刺激A是否呈现（1=呈现，0=未呈现）         |
| `Light_B`  | 视觉刺激B是否呈现（1=呈现，0=未呈现）         |
| `Sound_X`  | 听觉刺激X是否呈现（1=呈现，0=未呈现）         |
| `Sound_Y`  | 听觉刺激Y是否呈现（1=呈现，0=未呈现）         |
| `Stim_AX`  | 复合刺激AX是否呈现（1=呈现，0=未呈现）        |
| `Stim_AY`  | 复合刺激AY是否呈现（1=呈现，0=未呈现）        |
| `Stim_BX`  | 复合刺激BX是否呈现（1=呈现，0=未呈现）        |

## Q2 RW模型模拟（10分）

经典的RW模型（以下称为“特征RW模型”）假设实验动物学习单一刺激与食物奖赏之间的联结强度（$w_A$, $w_B$, $w_X$, $w_Y$），复合刺激的总联结强度为各单一刺激联结强度之和（如呈现复合刺激AX时，总联结强度为$w_A + w_X$）。

我们另外提出一种“组合RW模型”，实验动物将AX、AY、BX、BY视为相互独立的整体刺激，直接学习各复合刺激与食物奖赏的联结强度（$w_{AX}$, $w_{AY}$, $w_{BX}$, $w_{BY}$），更新规则与经典RW模型相同（即$\delta$规则）。

设所有联结强度初始值为0，学习率$\alpha$为0.1，试次序列与`data.csv`完全相同。

【绘图】请分别模拟“特征RW模型”与“组合RW模型”中联结强度随试次的更新过程并绘图：

- “特征RW模型”中，请画出（$w_A+w_X$, $w_A+w_Y$, $w_B+w_X$, $w_B+w_Y$）随试次变化的曲线；
- “组合RW模型”中，请画出（$w_{AX}$, $w_{AY}$, $w_{BX}$, $w_{BY}$）随试次变化的曲线。

## Q3 频率模型模拟（10分）

与RW模型基于$\delta$规则进行逐试次更新不同，我们提出一种基于客观共现频率的更新方式：条件刺激与食物奖赏的联结强度直接等于二者同时出现的频率。

以特征模型中的刺激A为例，假设前9个试次中，刺激A出现4次，其中3次有食物奖赏，则此时 $w_{A,\; t=9} = \frac{3}{4} = 0.75$。如果第10个试次中，A再次出现，同时有食物奖赏，则$w_{A, \; t=10} = \frac{4}{5} = 0.8$。

按照此更新方式，我们可以分别构建“特征频率模型”与“组合频率模型”。在“特征频率模型”中，复合刺激的总联结强度为各单一刺激联结强度的均值（如呈现复合刺激AX时，总联结强度为$\frac{w_A + w_X}{2}$）。

设所有联结强度初始值为0，试次序列与`data.csv`完全相同。

【绘图】请分别模拟“特征频率模型”与“组合频率模型”中联结强度随试次的更新过程并绘图，要求同Q2。

【文字解答】请对比频率模型和RW模型的模拟结果，比较二者联结强度变化趋势上的异同。

【文字解答】请尝试解释在“特征频率模型”中为何总联结强度要计算均值而非总和。
